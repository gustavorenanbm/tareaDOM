DOMDocument Object Model

Es el mapa de la página web creado por el navegador, a través de el JavaScritp puede modificar la pagina web

El DOM proporciona acceso a la página web




Document.body

Es una propiedad de JavaScritp que devuelve todo el contenido body de la página.
Document.title

Es una propiedad de JavaSript que devuelve o cambia el título de la pestaña
Document.images

Es una propiedad de JavaScript que devuelve un arreglo con las imágenes de la página.
Document.bgColor

Es un propiedad de JavaScript que cambia el color de fondo de la página(ahora CSS)




Características principales del DOM

Estrucura en Forma de árbol
Accesible mediante JavaScript
Interactivo
Actualizable en tiempo real


Objetivos del DOM

Presentar el contenido de forma estructurada
Facilitar el acceso de elementos a JavaScript
Interactivo
Permitir Manipulación Dinámica


Utilidad del DOM

Modificar el contenido
Crear interfaces interactivas
Actualizar partes específicas
Validar formularios
